# AI聊天客户端

* [返回上层目录](../multimodal-large-model.md)

既然你都有 **5090** 这种顶级配置了，一直盯着“黑框”对话其实有点浪费。为了更直观地看图片解析结果，我强烈建议你尝试以下两种方式：

- **Page Assist (浏览器插件):** 这是一个 Chrome/Edge 插件。它能直接连接你的本地 Ollama，给它套上一个类似 ChatGPT 的精美网页界面，支持**点击上传图片**，非常方便。
- **Chatbox / AnythingLLM:** 这种桌面客户端可以直接调用 Ollama。你只需要在界面里点击“加号”选图，解析结果（尤其是报表这种需要排版的内容）会以漂亮的 Markdown 表格形式展现出来。
